# Wallpapers

## Overview
Collection of 10 wallpaper images used by the wallpaper daemon (`hyprpaper` or `awww`) and lock screen (`hyprlock`).

## Directory
- `~/.dotfiles/wallpapers/Wallpapers/` — symlinked to `~/Wallpapers/` via stow.
- Formats: `.jpg`, `.jpeg`, `.png`.

## Notes
- The `rotate_wallpaper.sh` script cycles through these files.
- `hyprlock.conf` references `2825711.png` as the lock screen background.
- `hyprpaper.conf` references `wallhaven-x8jj9o_2560x1440.png` with `fit_mode = cover`.
